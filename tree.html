Which row breaks first?

The tree names row breaks first because it uses col-md. These columns stay in one row only on medium screens and larger (>= 768px). Below that size they stack. The letters row uses col-sm-2, so it stays in one row down to the small breakpoint (>= 576px) and breaks later.
